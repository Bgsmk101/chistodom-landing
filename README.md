# ЧистоДом — лендинг химчистки мебели

Адаптивный коммерческий лендинг для выездной химчистки мебели и ковров. Проект создан как демонстрационная работа для портфолио.

**Рабочая версия:** [Чистодом сайт](https://bgsmk101.github.io/chistodom-landing/#prices)

## Что реализовано

- адаптивная вёрстка для компьютеров, планшетов и телефонов;
- каталог услуг и цен;
- галерея результатов «до/после»;
- интерактивный калькулятор стоимости;
- этапы работы, преимущества, отзывы и FAQ;
- демонстрационная форма заявки;
- плавная навигация и доступные состояния фокуса;
- автоматическая публикация через GitHub Pages.

## Стек

- Next.js 16
- React 19
- TypeScript
- CSS без UI-библиотек
- GitHub Actions / GitHub Pages

## Локальный запуск

Понадобится Node.js 20.9 или новее.

```bash
git clone https://github.com/Bgsmk101/chistodom-landing.git
cd chistodom-landing
npm install
npm run dev
```

После запуска откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
```

Статическая версия будет создана в папке `out`.

> Название компании, телефон, цены, показатели и отзывы на странице являются демонстрационными.
